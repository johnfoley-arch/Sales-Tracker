# Sales Account Tracker

A lightweight sales account management tool for tracking action items and conversation notes — built as a single HTML file with no dependencies.

## Features

- **Account Sidebar** — Add, select, and manage sales accounts
- **Action Items** — Track tasks with due dates, checkboxes, and automatic archiving on completion
- **Conversation Notes** — Organize notes by conversation (e.g., per meeting or call)
- **Persistent Storage** — All data saved to browser localStorage

## Usage

Open `index.html` in any modern web browser. No build step or server required.

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- localStorage for persistence
- Google Fonts (Inter)
